# Security Policy

This is a public educational hobby repository for static webpages and offline learning games.

## Reporting

Please open an issue or contact the repository owner if you notice:

- Exposed credentials, API keys, tokens, or device secrets
- Unexpected external scripts or network calls
- Public files that accidentally include private details

## Project rules

- Static web pages should work without external services.
- Core2 games should remain offline unless the repository owner explicitly changes that direction.
- Do not commit `.env`, `secrets.h`, local configuration, build artifacts, serial logs, or private data files.
- Keep public content focused on the creative projects themselves.
